jSymbolic API

!
Index
to find Index within selector collection
@
Attribute
to add or change attributes
@x
Attribute
to remove attributes
&
Add style
to add style to elements
&+
Add Class
to add class to elements
&x
Remove Class
to remove style to elements
#
HTML5 Dataset
to manipulate HTML5 dataset functionality
